# PoinoTalk
![スクリーンショット](screenshot.png)

フォルマント合成を用いた日本語テキスト読み上げソフト  

## ブラウザで使う
PoinoTalk 最新版  
https://koharuyuzuki.github.io/poinotalk/latest/  

PoinoTalk バージョン一覧  
https://koharuyuzuki.github.io/poinotalk/versions/  

## Q&A
Q&A.txt をご参照ください  

## ビルド
```
$ bun install
$ bun run build.ts
```
